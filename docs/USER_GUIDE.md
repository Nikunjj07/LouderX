# Sydney Events Aggregator - User Guide

## Welcome!

This guide will help you get the most out of the Sydney Events Aggregator - your one-stop destination for discovering events in Sydney.

---

## Getting Started

### Accessing the Website

1. Open your web browser
2. Navigate to the website URL
3. The homepage will load with current events

---

## Finding Events

### Browsing Events

When you first visit the site, you'll see:
- A grid of upcoming events in Sydney
- Each event card shows:
  - Event title
  - Date and time
  - Location
  - Brief description
  - Source website
  - "Get Tickets" button

### Understanding Event Cards

**Event Image**: Visual representation of the event (if available)

**Event Title**: Name of the event

**Date & Time**: When the event occurs (in Sydney timezone)

**Location**: Venue or area where event takes place

**Description**: Brief overview of what to expect

**Source**: Website where this event was found

---

## Using Filters

### Search by Keyword

1. Locate the **Search** box at the top
2. Type keywords (e.g., "festival", "music", "food")
3. Results filter automatically as you type
4. Events matching your search appear instantly

**Search Tip**: Search looks through event titles, descriptions, and locations

### Filter by Date

Use the **Date** dropdown to see:
- **All Dates**: Shows all upcoming events
- **Today**: Events happening today
- **Tomorrow**: Events happening tomorrow
- **This Week**: Events in the next 7 days
- **This Month**: Events in the next 30 days

### Filter by Location

1. Open the **Location** dropdown
2. Select a specific venue or area
3. Only events at that location will display

**Location Tip**: Locations are automatically populated from available events

### Reset Filters

Click the **Reset Filters** button to:
- Clear all active filters
- Return to showing all events
- Reset search box

## Getting Tickets

### How to Subscribe and Get Tickets

1. **Find an event** you're interested in
2. **Click "Get Tickets"** on the event card
3. A popup window will appear

### Email Subscription Form

**Why we ask for your email**:
- Get notified about event updates
- Receive information from event organizers
- We never spam or share your email

**How to complete the form**:

1. **Enter your email**:
   - Must be a valid email format
   - Example: yourname@example.com

2. **Check the consent box**:
   - Required to receive event updates
   - You control your subscriptions

3. **Click "Subscribe & Get Tickets"**:
   - Your email is saved
   - You'll be redirected to buy tickets

### After Subscribing

- The popup closes automatically
- A new tab opens with the ticketing website
- You can purchase tickets directly from the source

## Frequently Asked Questions

### Q: How often are events updated?
**A**: Events are automatically scraped from source websites every 6 hours, so you always see fresh content.

### Q: Are these events free?
**A**: Some are free, some require tickets. Check the ticketing website after subscribing for pricing details.

### Q: Will I get spam emails?
**A**: No. We only store your email to track your interest in specific events. We don't send emails ourselves (yet).

### Q: Can I unsubscribe?
**A**: Yes, contact us with your email address and we'll remove your subscriptions from our database.

### Q: What if an event is sold out?
**A**: When you click through to the ticketing website, they'll show current availability.

### Q: Are events verified?
**A**: Events are automatically scraped from reputable Sydney event websites. Always verify details on the official ticketing site.

### Q: What if I find incorrect information?
**A**: Event information comes from source websites. Please verify details on the official event page.

### Q: Which websites do you scrape?
**A**: Currently:
- Time Out Sydney
- Eventbrite Sydney
- What's On Sydney

### Q: Can I suggest a new event source?
**A**: Yes! Contact the site administrators with your suggestions.

### Q: Do you have a mobile app?
**A**: Not yet, but the website works perfectly on mobile browsers.

## Troubleshooting

### Events Not Loading

**If you see "Loading..." for a long time**:
1. Check your internet connection
2. Refresh the page
3. Try clicking the "Retry" button if it appears
4. Clear your browser cache

### Filter Not Working

**If filters don't respond**:
1. Refresh the page
2. Try clicking "Reset Filters"
3. Make sure JavaScript is enabled
4. Try a different browser

### Can't Get Tickets

**If the email form doesn't work**:
1. Check your entered email is valid
2. Make sure you checked the consent box
3. Try refreshing and submitting again
4. Check if you have an ad blocker interfering

### Website Looks Broken

**If the layout appears wrong**:
1. Make sure you're using a modern browser
2. Update your browser to the latest version
3. Try on a different device
4. Clear browser cache and reload
